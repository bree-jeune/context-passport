# Quick-Start Prompts

Copy-paste these at the start of new sessions.

---

## Standard Resume

```
Read my Context Passport at [GITHUB_RAW_URL]/PASSPORT.md and continue where we left off. Don't reintroduce yourself. Pick up the thread.
```

---

## With Specific Focus

```
Read my Context Passport at [URL]/PASSPORT.md. Today we're focusing on [PROJECT/TASK]. Continue from my last Context Stack entry.
```

---

## Full Context Paste

If you can't link to GitHub, paste the PASSPORT.md content directly:

```
Here's my Context Passport - read it and continue where we left off:

[PASTE PASSPORT CONTENT HERE]

---

Don't reintroduce yourself. I know you're Claude. Let's pick up the thread.
```

---

## After Long Break

```
Read my Context Passport at [URL]. It's been a while since we worked together. Briefly confirm you've got my context, then let's resume.
```
